# NorthHabit

A premium ambient productivity and habit-tracking experience designed for deep focus, intentional routines, and sustainable digital wellness.

🌐 Live Website: https://northhabit.pages.dev

---

# Preview

NorthHabit combines:

* Habit tracking
* Pomodoro focus systems
* Ambient productivity design
* Analytics dashboards
* Immersive UI interactions
* PWA capabilities

into a calm and emotionally engaging productivity environment.

The project focuses on making productivity feel immersive rather than stressful.

# Design Philosophy

NorthHabit intentionally avoids:

* aggressive gamification
* dopamine-heavy UX
* cluttered interfaces
* corporate productivity aesthetics

Instead, the experience focuses on:

* calmness
* immersion
* intentionality
* sustainability
* emotional design
